# Portfolio Website — Maggie Sievers

## About This Project

This project portfolio is a reflection of my web design skills that I used in ITD 210. I built this site to practice turning ideas into real, functional web pages. I chose the topic of marketing because marketing is what helps companies to grow and I hope to work in that field in order to help them do so. Each page of my website represents my progress in creating clean, user-friendly experiences and organizing code in a way that makes sense for the user.

## Technologies Used

* HTML5
* CSS3 (Flexbox, Grid, Responsive Design)
* JavaScript
* Tailwind CSS
