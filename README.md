Project Description

This project is a fully responsive multi-page website I built as my final capstone for ITD 210. The goal of this project was to bring together everything I learned throughout the course, including layout design, responsiveness, accessibility, and JavaScript interactivity, into a complete and cohesive website.

I focused on creating a consistent visual design system across all pages while making sure the site works smoothly on mobile, tablet, and desktop devices. A major part of my process was refining the user experience so the navigation, structure, and interactions feel intuitive and polished.

Overall, this project reflects my growth in HTML, CSS, and JavaScript, especially in building structured layouts, improving accessibility, and adding meaningful interactivity.

Technologies Used
HTML5
CSS3 (Flexbox and Grid)
JavaScript (DOM manipulation and interactivity)
Responsive design using media queries
Accessibility practices aligned with WCAG 2.1 AA standards
Features
Five fully linked pages: Home, About, Gallery, Media, Contact
Fully responsive layout for mobile (375px), tablet (768px), and desktop (1280px)
Consistent visual design system across all pages
Site-wide interactive feature (such as dark mode toggle, back-to-top button, or sticky navigation)
Scroll-based animations and transitions
Contact form with validation
Embedded media content including images and video
Keyboard-accessible navigation across all pages
Accessibility
WCAG 2.1 AA color contrast compliance across the entire site
Full keyboard navigation support (Tab, Shift+Tab, Enter)
Alt text included for all images
Proper form labels for screen readers
Visible focus states for interactive elements
What I Learned

Through this project, I learned how to:

Plan and build a complete multi-page website from start to finish
Create responsive layouts that adapt smoothly across screen sizes
Use JavaScript to add meaningful interactivity and improve user experience
Apply accessibility standards to make a site more inclusive
Maintain consistency in design across multiple pages
Debug and improve HTML, CSS, and JavaScript in a real project environment

One of the biggest takeaways for me was understanding how small design and structure decisions can significantly affect usability and overall user experience.
