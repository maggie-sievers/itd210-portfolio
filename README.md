Portfolio Website Milestone 5 — Maggie Sievers
About This Project

This portfolio website was created as part of ITD 210 Web Development at Northern Virginia Community College. It demonstrates my continued growth in web design and development with a focus on building responsive, accessible, and interactive user experiences.

For this milestone, I developed a fully functional Contact page designed to simulate a real-world client inquiry experience. The page focuses on usability, accessibility, and interactivity while maintaining a consistent visual identity across the full website.

The contact page includes a validated form connected to Formspree, interactive JavaScript form validation, and an expandable FAQ accordion to improve user engagement. This milestone also emphasizes accessibility standards such as keyboard navigation, ARIA attributes, and clear error messaging.

Each page continues to improve in layout design, code organization, responsiveness, accessibility, and interactivity based on instructor feedback from previous milestones.
