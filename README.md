# Portfolio Website Milestone 4 — Maggie Sievers

## About This Project

This portfolio website was created as part of ITD 210 Web Development at Northern Virginia Community College. It demonstrates my continued growth in web design and development with a focus on building responsive, accessible, and interactive user experiences.

For this milestone, I designed a dedicated Media page that integrates video, audio, and animation into a cohesive and engaging storytelling experience. The goal was to make the page feel dynamic and interactive rather than static.

Each page continues to improve in layout design, code organization, responsiveness, accessibility, and interactivity based on instructor feedback from previous milestones.


## Technologies Used

- HTML5  
- CSS3 (Flexbox, Grid, Animations, Responsive Design)  
- JavaScript (DOM manipulation, interactivity, event handling)  


## What’s Included in This Milestone (Media Page Features)

- Responsive HTML5 video with poster image and native controls  
- Embedded audio element with full transcript for accessibility  
- CSS animation (typewriter effect on hero text)  
- Scroll-reveal animation for interactive section loading  
- JavaScript-powered mobile navigation (hamburger menu)  
- JavaScript media interaction support (play/pause video control)  
- Interactive media layout using cards and hover effects  
- Accessibility improvements including transcripts, readable contrast, and reduced motion support  
- Mobile-first responsive design across all media components  


## Accessibility Features

- Video and audio include transcripts or text summaries  
- Semantic HTML structure used throughout  
- Keyboard-friendly navigation support  
- Reduced motion support using prefers-reduced-motion media query  
- Clear visual hierarchy and readable typography  


## AI / Tool Assistance Disclosure

Some debugging support, JavaScript assistance, and structural guidance were developed with help from ChatGPT (OpenAI). This included help refining interactive media controls, improving accessibility structure, and organizing code comments.

All final design decisions, customization, styling, and implementation were completed by me.
