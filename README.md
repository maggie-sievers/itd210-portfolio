# Portfolio Website Milestone 3— Maggie Sievers

## About This Project

This portfolio website was created as part of ITD 210 Web Development at Northern Virginia Community College. It showcases my growth in web design and development, focusing on building responsive, accessible, and user-friendly web pages.

I chose a marketing-focused theme because marketing plays a key role in helping businesses grow. My goal is to develop skills that allow me to create effective digital experiences for real-world use.

Each page of this site demonstrates progress in layout design, code organization, responsiveness, and interactive elements.


## Technologies Used

- HTML5  
- CSS3 (Flexbox, Grid, Responsive Design)  
- JavaScript  


## What’s Included in This Milestone

- Responsive gallery/portfolio grid layout  
- Interactive hover overlays on project cards  
- Image modal/lightbox feature  
- Filter buttons for project categories  
- Improved accessibility (alt text, keyboard support, reduced motion handling)  
- Refined and responsive CSS structure across all pages  


## AI / Tool Assistance Disclosure

Some HTML and CSS structure, layout improvements, and debugging support were developed with assistance from ChatGPT (OpenAI). All final design decisions, customization, and implementation were completed by me.
