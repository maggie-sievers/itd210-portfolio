Project Description

This project is a fully responsive multi-page website built as a final capstone for ITD 210. It includes a consistent design system, interactive features, and responsive layouts optimized for mobile, tablet, and desktop devices.

The site demonstrates skills in HTML, CSS, and JavaScript, including layout design, accessibility practices, and client-side interactivity.
